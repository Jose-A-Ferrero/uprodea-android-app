# 📱 UPRODEA – App Android para Gestión de Protectora

Aplicación móvil desarrollada en Kotlin para la gestión interna de una protectora de animales.
Actualmente en uso real.

---

## 🚀 Problema que resuelve

La gestión manual de animales y vacunas dificultaba el seguimiento de estos y es con la aplicacion como hemos dado solucion a este problema.

La aplicación permite centralizar y digitalizar toda la información desde el móvil.

---

## 🛠 Tecnologías utilizadas

- Kotlin
- Android Studio
- SQLite
- Gestión de base de datos local
- Persistencia de datos

---

## 🎯 Funcionalidades principales

- Registro de animales
- Gestión de vacunas
- Edición y actualización de datos
- Almacenamiento persistente en base de datos local
- Interfaz optimizada para uso interno

---

## 🧠 Enfoque técnico

- Estructuración organizada del proyecto
- Separación de lógica y presentación
- Manejo eficiente de SQLite
- Orientación a estabilidad y usabilidad

---
